System at a glance

Pulse HK exposes a single A2A endpoint, orchestrated by a supervisor that delegates to specialised sub‑agents (BrandFinder, OfferFinder, GeneralQA). Long‑term memory is powered by Mem0, and every execution is fully observable via AgentMonitor. Below you'll find a complete architectural walkthrough.

Design Decisions

Six architectural pillars

🎯 Task ID as @MemoryId
Each request isolated planning window.
💬 Memories into prompt
Prevents LLM hallucinating agent names.
🧹 Window sizes by contract
Deterministic token budgets.
🔗 Dual identity propagation
ThreadLocal + AgenticScope.
🧪 AgenticScopeAccess
Avoids “multiple agent methods” error.
🌊 Topic‑segmented metadata
Clean retrieval per domain.
