System at a glance

Pulse HK exposes a single A2A endpoint. Inside, a supervisor orchestrates BrandFinder, OfferFinder, GeneralQA, while Mem0 stores cross‑session memories. Every request is monitored via AgentMonitor.

C4 · System Context

Topology & wiring

The Spring Boot application intercepts A2A calls, resolves identity, injects memory context, and delegates to the supervisor. Sub‑agents are pure Java interfaces — LangChain4j generates proxies.

Long‑Term Memory

Mem0 semantic loop

Memories are stored with topic metadata (brands, preferences, general). At retrieval, three parallel searches enrich the prompt. The chat history never sees system‑level memory messages → prevents hallucination.
